Float vs Decimal 
6 
⚠  Floating-Point 
Stores numbers in binary 
Causes rounding errors 
0.1 + 0.2 = 0.30000000000000004 
✓ DECIMAL 
Stores numbers in exact base-10 
No rounding errors 
DECIMAL(10,2)  
10 = precision → total number of 
digits allowed 
2 = scale → number of digits after 
the decimal point 
E.g., 12345678.90 
When to Use: Money, Accounting, Precise Measurements 
Use DECIMAL when accuracy matters 
Try it in Python.

SELECT Statement Overview 
SELECT  <list of column expressions> 
FROM  <list of tables and join operations> 
WHERE  <list of logical expressions for rows> 
ORDER BY  <list of sorting specifications> 
 
• Column expression: combination of columns, constants, operators, and functions 
– Example: FacSalary * 1.1 
 
• Logical expression: conditions connected by AND, OR, and NOT 
– Example: OffTerm = 'FALL' AND OffYear = 2016

--- PAGE 33 ---
Order by  
• ORDER BY column_name DESC|ASC 
• Use DESC to specify decreasing order 
• Use ASC to specify increasing order 
• If not given, default is ASC 
 
SELECT payment_id, customer_id, rental_id,  
 amount, payment_date  
FROM sakila.payment 
WHERE amount> 0 
ORDER BY payment_date;

Distinct 
• The DISTINCT keywoard is used to return 
only distinct (different) values. 
• Inside a table, a column often contains 
many duplicate values; and sometimes 
you only want to list the different 
(distinct) values. 
 
SELECT amount 
FROM sakila.payment 
ORDER BY amount DESC; 
SELECT DISTINCT amount 
FROM sakila.payment 
ORDER BY amount DESC;

Limit 
• LIMIT keyword can be used to specify the number of records to 
return. 
• LIMIT is useful on large tables. Returning a large number of records 
can impact query performance. 
• In some databases the keyword is TOP 
SELECT DISTINCT amount 
FROM sakila.payment 
ORDER BY amount DESC 
LIMIT 5; 
SELECT *  
FROM sakila.payment 
LIMIT 3
