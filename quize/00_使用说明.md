# Quiz 复习包使用说明

本文件夹根据课程 PPT 与 `sample_questions.pdf` 整理，面向 quiz/exam 复习。

建议阅读顺序：

1. `01_四章章节要点.md`：先建立全局知识框架。
2. `02_考试简答题list.md`：按简答题方式背诵和练习。
3. `03_SQL编写代码题答题步骤.md`：按步骤训练 SQL 题，尤其是 sample question 的题型。
4. `04_思维导图.md`：用 Mermaid 思维导图串联四章逻辑。
5. `extracted_text/`：PDF 原文抽取结果，方便回查 PPT 表述。

考试重点判断：

- 概念题大概率围绕 primary key / foreign key、OLTP vs OLAP、data warehouse、ETL、data staging area、star schema、dimension/fact table、surrogate key、normalization。
- 代码题大概率围绕 `CREATE TABLE`、`SELECT`、`WHERE`、`ORDER BY`、`LIMIT`、`JOIN`、`GROUP BY`、`HAVING`、subquery/CTE、window function 或“每个客户最近三笔交易”这类每组 top N。
- SQL 默认按 MySQL 风格准备，因为 PPT 多处使用 MySQL 函数与 `LIMIT`。
